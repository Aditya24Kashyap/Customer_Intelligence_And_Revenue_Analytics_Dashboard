# Customer_Intelligence_And_Revenue_Analytics_Dashboard
Customer Intelligence & Revenue Analytics Dashboard is a data-driven project designed to monitor and analyze key business performance metrics. It highlights essential KPIs such as total revenue, average order value (AOV), and repeat customer percentage to provide a clear view of business health. The dashboard also includes customer segmentation to identify different customer groups based on behavior and purchasing patterns. By analyzing factors like customer retention and the impact of discounts on sales, the project delivers actionable insights that support smarter decision-making and revenue optimization. Using Python, SQL, PowerBI.
