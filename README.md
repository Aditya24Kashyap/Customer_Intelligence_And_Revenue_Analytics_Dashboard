# 📊 Customer Intelligence & Revenue Analytics Dashboard

## 🚀 Project Overview

This project demonstrates an end-to-end Business Intelligence pipeline that transforms raw customer shopping data into actionable insights. It showcases how data can be cleaned, analyzed, and visualized to support data-driven business decisions.

**Solution integrates:**

* Python → Data cleaning & feature engineering
* SQL → Business analysis queries
* Power BI → Interactive dashboard & visualization

---

## 🎯 Objective

Analyze customer behavior and answer key business questions:

* Who are the most valuable customers?
* Do discounts impact spending?
* Are subscription users more profitable?
* Which categories and regions drive revenue?

---

## 🧰 Tech Stack

* Python (Pandas, NumPy) – Data preprocessing
* SQL (PostgreSQL/MySQL) – Business queries
* Power BI – Dashboard & visualization
* Jupyter Notebook – Development environment

## 🔄 Workflow

### 1️⃣ Data Preparation (Python)

* Loaded dataset using Pandas
* Handled missing values (median imputation for ratings)
* Renamed columns for consistency
* Created new features:

  * `age_group`
  * `purchase_frequency_days`

### 2️⃣ Data Analysis (SQL)

* Revenue by gender
* Top-rated products
* Customer segmentation (New / Returning / Loyal)
* Subscription vs revenue analysis
* Discount impact on purchases
* Top products per category

### 3️⃣ Dashboard Development (Power BI)

**🔝 KPI Cards**

* Total Revenue
* Total Customers
* Average Order Value (AOV)
* Repeat Customer %
* Average Rating

**📊 Visualizations**

* Revenue by Category
* Customer Segmentation
* Age Group vs Revenue
* Subscription vs Revenue
* Discount Impact Analysis
* Location-wise Sales

**🎛 Filters**

* Gender
* Category
* Season
* Subscription Status

---

## 💡 Key Insights

* Loyal customers contribute significantly higher revenue
* Subscription-based customers show higher engagement
* Discounts do not reduce spending, indicating effective promotions
* Young and middle-aged customers drive the majority of revenue
